Guia de Programação Web com HTML

Neste artigo, vamos explorar as principais tags HTML e suas funcionalidades.

Tags de Estrutura Básica

Vamos começar pelas tags básicas que formam a estrutura de uma página HTML:

<!DOCTYPE html>: Define o tipo de documento como HTML5.
<html>: Tag raiz que envolve todo o conteúdo da página.
<head>: Contém informações meta sobre a página.
<meta>: Fornece metadados, como a codificação de caracteres.
<title>: Define o título da página.
<body>: Contém o conteúdo visível da página.

Tags de Texto e Cabeçalhos

Formate e organize o texto na página com estas tags:

<h1> a <h6>: Cabeçalhos de diferentes níveis.
<p>: Parágrafo de texto.
<br>: Quebra de linha.
<strong>: Texto em negrito.
<em>: Texto enfatizado (itálico).

Tags de Links e Imagens

Adicione links e imagens às suas páginas:

<a>: Cria um link para outra página ou recurso.
<img>: Insere uma imagem na página.

Tags de Formulário

Crie interatividade com formulários usando estas tags:

<form>: Cria um formulário para entrada de dados.
<input>: Cria campos de entrada.
<textarea>: Cria uma área de texto de várias linhas.
<select>: Cria uma lista suspensa.
<label>: Rótulo para elementos de formulário.
<button>: Cria botões interativos.

Tags de Tabelas

Organize dados em tabelas usando essas tags:

<table>: Cria uma tabela.
<tr>: Cria uma linha na tabela.
<td>: Cria uma célula de dados.
<th>: Cria uma célula de cabeçalho.
<thead>: Define o cabeçalho da tabela.
<tbody>: Define o corpo da tabela.
<tfoot>: Define o rodapé da tabela.

Tags de Mídia e Estilo

Enriqueça o conteúdo da página com mídia e estilo:

<audio>: Incorpora áudio na página.
<video>: Incorpora vídeos na página.
<source>: Define a origem de mídia.
<style>: Adiciona estilos CSS no documento.
<link>: Conecta um documento externo de estilo CSS.

Tags Semânticas e Outras

Utilize tags semânticas e outras funcionalidades:

<nav>: Define uma seção de navegação.
<aside>: Define um conteúdo lateral.
<mark>: Destaca um trecho de texto.
<time>: Define uma data ou hora.
<progress>: Exibe o progresso de uma tarefa.
<hr>: Cria uma linha horizontal.
